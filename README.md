# 🦅 Benfica AI Ticket Monitor

## Product Manager Portfolio Project by leifwhitman

A sophisticated AI-powered ticket monitoring system demonstrating modern product management skills.

### 🚀 Features
- AI-powered real-time web search
- Secure serverless backend
- Responsive design
- Real-time notifications

### 🛠️ Tech Stack
- Frontend: HTML, Tailwind CSS, JavaScript
- Backend: Netlify Functions (Node.js)
- AI: DeepSeek API with web search

### 📦 Deployment
This project is designed to be deployed on Netlify with environment variables for secure API key management.
